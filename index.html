Vodič za početnike koristi gotove kôdove web stranica koje možete kopirati u editor i isprobati kako funkcionira html skriptni jezik. Ponekad može za početnike izgledati da je html kôdiranje složen posao. Naravno svaki je početak težak no slijedeći lekcije i u njima primjere za pojedine html elemente korak po korak moći ćete razumjeti logiku html kôda. HTML kôd omogućava unos sadržaja web stranice dok css daje mogućnost njezinog vizualnog oblikovanja. No o tome kasnije.

Zašto HTML?

Riječ "hyper" znači suprotno od "linear". Primjer teksta koji je linearan je tekst u knjizi. Nema mogućnosti skokova iz jednog dijela teksta na drugi dio koji se nalazi bilo u toj knjizi ili u nekoj drugoj knjizi. Za razliku od toga html omogućava izravno prebacivanje s jednog mjesta na drugo, kako unutar iste web stranice tako i na neku drugu web stranicu. Riječ "tekst" označava da su html kôdovi napisani kao tekst bez formatiranja. Oznaka "Mark-up Language" govori nam da označavanje html kôdova omogućava web pregledniku prikazivanje različitih elemenata na web stranici kao što su naslovi, grafički i simbolički popisi, slike i drugi.

HTML naslovi:

HTML razlikuje 6 razina naslova u tekstu. Naslovi se međusobno razlikuju po veličini fonta. Najveći je glavni naslov (naslov veličine 1) odnosno element h1 a najmanji je naslov veličine 6 odnosno element h6. Potrebno je razlikovati naslove h1 do h6 unutar teksta web stranice od naslova web stranice title koji se ispisuje na kartici web stranice.

HTML kodovi i web dizajn:

Želimo li umetnuti tekst unutar web stranice najjednostavnije je koristiti za to html elemente odlomka: p (engl. paragraphs). Svaki html odlomak počinje u novom retku. Želimo li oblikovati tekst koristimo oznake elemenata za podebljan ili nakošen tekst, tekst u eksponentu ili indeksu i druge.

HTML podebljan tekst: Strong - omogućava podebljavanje dijela teksta.

HTML nakošen tekst : Italic - možemo nakositi dio teksta korištenjem elementa.
